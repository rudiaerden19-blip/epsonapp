# Epson app

Eigen repository voor Epson-gerelateerde code en documentatie.

## USB bonprinter op Windows (kassa-PC)

Zie **`usb-print-bridge/`**: lokale server op `127.0.0.1:3001` → Epson via USB-COM-poort. Dezelfde API als de bestaande print-server op poort 3001.

Root **`index.html`** is een placeholder voor een aparte landingspagina.
